# IOT-BASED-SMART-WATER-QUALITY-MONITORING-SYSTEM
IoT-based Smart Water Quality Monitoring System using ESP32/NodeMCU, Firebase Realtime Database, and a web dashboard to monitor pH, TDS, Turbidity, and Temperature in real time. The system provides cloud-based data storage, live visualization, sensor calibration, and remote water quality monitoring for smart and sustainable water management.
🚀 Features
Real-time monitoring of water quality parameters
pH, Temperature, Turbidity, and TDS sensing
Firebase Realtime Database integration
Interactive web dashboard with live gauges and charts
Sensor calibration panel for accurate measurements
Online/Offline device status monitoring
Historical data visualization using Chart.js
Cloud-based remote monitoring from any location
🛠️ Technologies Used
ESP32 / NodeMCU
Arduino IDE
Firebase Realtime Database
HTML, CSS, JavaScript
Chart.js
JustGage
IoT Sensors (pH, TDS, Turbidity, Temperature)
🎯 Applications
Drinking Water Quality Monitoring
Smart Water Management Systems
Industrial Water Monitoring
Aquaculture and Fish Farming
Environmental Monitoring
Research and Educational Projects
📌 Project Objective

The primary objective of this project is to provide a low-cost, scalable, and real-time solution for monitoring water quality. By integrating IoT technology with cloud-based data storage and visualization, the system enables users to remotely track water conditions and take timely actions when water quality parameters exceed safe limits.
